# React + Vite
This code generates my portfolio website!

## Requirements

- Nodejs latest version

## Installation

1. Clone the repository:
   
    ```
    git clone  https://github.com/Ge0rge-Dim/my-portfolio
    ```
2. Navigate to the directory:
   
    ```
    cd my-portfolio
    ```
3. Run the program:
   
    ```
    npm run dev
    ```
